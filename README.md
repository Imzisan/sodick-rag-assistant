# sodick-rag-assistant
Retrieval-Augmented Generation (RAG) system that indexes and queries Sodick website content to deliver contextual AI responses

# How to run?
### STEPS:

Clone the repository

```bash
Project repo: https://github.com/
```
### STEP 01- Create a conda environment after opening the repository

```bash
conda create -n sodibot python=3.10 -y
```

```bash
conda activate sodibot
```


### STEP 02- install the requirements
```bash
pip install -r requirements.txt
```
